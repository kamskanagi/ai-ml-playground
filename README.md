# AI/ML Playground

A collection of machine learning and AI projects covering computer vision, RAG, model fine-tuning, speech processing, and agentic systems.

## Projects

### 1. **Potato Disease Classification** 
Deep learning application for agricultural disease detection using convolutional neural networks.

### 2. **End-to-End Medical Chatbot** 
RAG-powered conversational AI for medical information with production safety features.

### 3. **Fine-tuning with Unsloth** 
Educational notebook demonstrating parameter-efficient fine-tuning on free GPU resources.

### 4. **AI Transcription & Cleanup Tool** 
Full-stack audio transcription with LLM-powered text refinement.

### 5. **Local AI Agent with phi3 (SLM)**
Interactive CLI agent running entirely locally via [Ollama](https://ollama.com) and phi3. No cloud APIs or API keys required. Uses a prompt-based ReAct loop with tools for calculation, web search, and summarisation.

### 6. **Multi-Agent System**
LangGraph-based pipeline that routes queries through a Planner → Worker → Reviewer loop with automatic revision cycles. Powered by Llama 3.3 70B via Groq.

### 7. **Manufacturing Defect Classifier**
ResNet-50 transfer learning pipeline for visual quality control. Classifies product images as good or defective, handles class imbalance with weighted loss, and runs single-image inference. Supports CUDA, Apple Silicon (MPS), and CPU.
